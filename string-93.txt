* Tor Project Web / Tor Browser User Manual / string 93 *

ORIGINAL TEXT
-------------
Tor Browser includes some circumvention tools for getting around these blocks.
These tools are called “pluggable transports”.

TRANSLATION
Tor Browser включает в себя некоторые инструменты обхода этих блоков.
Эти инструменты называются "подключаемые транспорты".
